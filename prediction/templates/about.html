Heart Disease Prediction System

About This Project

This web application predicts the likelihood of heart disease using advanced machine learning models such as Logistic Regression and Linear Regression.

Users can register, login, and input their health data including age, sex, chest pain type, cholesterol level, blood pressure, ECG results and more.

The system then provides predictions and helps in understanding potential risks.
